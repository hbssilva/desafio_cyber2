# desafio_cyber2
Entrega Desafio Cyber2

# Etapas do Keylogger:

# 1 - Ficará executando em segundo plano;
# 2 - Toda vez que o usuario digitar uma tecla, o programa irá capturar essa tecla; 
# 3 - O que for digitado, será gravado em um arquivo .txt;
# 4 - O arquivo vai mostrar tudo o que foi digitado, de forma sequencial;
# 5 - Enviar por e-mail dados do que foi capturado ao atacante;
